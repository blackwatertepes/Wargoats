Context Optional: Birthday Cake © 2011

Select your favorite style of birthday cake. Then your friends can see which type of cake you'd prefer to receive on your birthday. The Facebook application was developed for Context Optional using Rails and the Facebook API. The Faceball gem was used, which is a proprietary Facebook API library. It closely resembles Koala, which is the open-source version, released by Context Optional.

Responsibilities

  • Design
  • HTML/CSS
  • Javascript (jQuery)
  • Rails 2.7
  • mySQL
  • Facebook API (Faceball library, similar to Koala)

Bravo: Make Me A Model © 2009

Upload an image of yourself, and touch up blemishes using a application in your web browser. The application was written entirely in Actionscript 3.0, with the obvious exception of uploading, and saving images to the server. Algorithms for smoothing skin blemishes and removing red eye were particularly challenging, and were accomplished by gathering pixel color in the surrounding area, and recoloring pixels over the cursor.

Responsibilities

  • Actionscript 3.0

Toyota Prius: Touch and Go © 2009

A Facebook game that mirrored Simple Simon. The application was written in Actionscript 3.0, with the exception of saving and retreiving game scores, and friend information. The game presented the user with a series of colors, to which they had to repeat the color pattern to earn points.

Smart.fm: Brain Speed © 2009

Take quizzes on how well you know your Facebook friends. Quizzes were created automatically, using the users friends profile information. The application was written entirely in Actionscript 3.0, and gathered data about the users friends interests using the Flash version of the Facebook API.

Responsibilities

  • Design
  • Actionscript 3.0
  • Facebook API (Flash API)

Travel Channel: Kidnap! © 2008

Kidnap your friends to exotic locales. Then have them escape by answering quiz questions about the destination they were kidnapped to. The application was written entirely in Actionscript 3.0, with the exception of saving and retrieving user scores. This particularly popular Facebook application garnered over 1 million Facebook users, but sadly is no longer running.

Responsibilities

  • Actionscript 3.0
